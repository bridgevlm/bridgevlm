Implementation Code

BridgeVLM bridges Vision Language Models (VLM) with CLIP embeddings through learnable query tokens and cross-attention mechanisms. The algorithm processes anchor images through a VLM, refines queries via transformer decoder, and projects to CLIP space for contrastive learning with ground truth images.
